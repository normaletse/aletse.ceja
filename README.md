Análisis de Ventas de Videojuegos – Ice Store
Contexto del proyecto
Este proyecto analiza datos históricos de ventas de videojuegos de una tienda online global llamada Ice, con el objetivo de identificar patrones que determinan si un videojuego es exitoso o no.

El dataset incluye información como:

Ventas por región

Género del videojuego

Plataforma (PlayStation, Xbox, etc.)

Reseñas de usuarios y expertos

Año de lanzamiento

Clasificación por edades (ESRB rating)

Objetivo del análisis
Detectar características comunes en los videojuegos exitosos para:

Identificar proyectos prometedores

Apoyar la toma de decisiones estratégicas

Optimizar campañas de marketing

Planificar estrategias de ventas futuras

Enfoque del proyecto
Se utilizó información histórica hasta 2016 para simular la planeación de campañas publicitarias para el año siguiente, aplicando análisis de datos como base para la toma de decisiones.

Valor del proyecto
Este análisis permite transformar datos en información estratégica, demostrando habilidades en:

Exploración de datos (EDA)

Análisis de patrones

Interpretación de información

Pensamiento analítico

Enfoque en negocio

Data-driven decision making
